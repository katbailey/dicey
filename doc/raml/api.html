Decision

/decision get

get

Get a decision about content to show

get /decision

Get a decision about content to show

  • Request
  • Response

Query Parameters

  • cid: required (string)

    The container identifier for which we need to make a decision

  • pid: required (string)

    The unique hash of the user for whom we are making a decision for so we can identify this user later

      "minItems": 1,

          "thumbnail_url": {
            "pattern": "^(https?://)?([\\da-z\\.-]+)\\.([a-z\\.]{2,6})([/\\w \\.-]*)*/?$",
            "type": "string"
          },
          "type": {
            "pattern": "^[a-z0-9_]{1,128}$",

          },
          "url": {
            "pattern": "^(https?://)?([\\da-z\\.-]+)\\.([a-z\\.]{2,6})([/\\w \\.-]*)*/?$",

          }

    }

  ]
